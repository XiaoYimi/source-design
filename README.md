<h1 align="center">源设计</h1>

## 简介
- 自`2023.06.01`起，往返来回考察、总结大厂开发全流程程序与经验。
- 以`2024.12.01`起，手写源码原理，学习开发思想，总结开发文档。


## 演武场
- 下载项目源码
```bash
git clone https://github.com/XiaoYimi/source-design.git
```

- 安装项目依赖
```bash
yarn
```

- 运行项目
```bash
yarn dev
```
